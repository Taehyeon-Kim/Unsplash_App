# Unsplash_API_TEST_APP
💁🏻  Unsplash API를 통한 API 통신 연습

<br />

- **공부 및 개발 기간 : 2021.02.08 ~ 2021.02.16**

<br><br>

## 👨🏻‍💻 핵심 기능

✅ 메인 화면 사진 30장 랜덤 노출  
✅ 사진 검색  
✅ 사진 보기  
✅ 사진 다운로드  
✅ 사진 색상 추출  
✅ Unsplash 프로필 로드  
✅ 좋아요한 사진 리스트 확인

## 📄 라이브러리

| 라이브러리(Library) | 목적(Purpose)    | 버전(Version) |

| Kingfisher           | 이미지 캐싱 및 다운로드        | 6.0           |
| KakaoSDKCommon      | 카카오 공통 모듈 |               |
| KakaoSDKAuth        | 카카오 로그인    |               |
| KakaoSDKUser        | 사용자 관리    |               |

## 📜 커밋 컨벤션

```
  - init : 초기화
  - add : 파일 추가
  - feat : 기능 추가
  - delete : 기능 삭제
  - update : 기능 수정
  - fix : 버그 수정
  - refactor: 리팩토링
  - style : 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없음)
  - docs : 문서 (문서 추가(Add), 수정, 삭제)
  - chore : 기타 변경사항 (빌드 스크립트 수정, 에셋 추가 등)
```
