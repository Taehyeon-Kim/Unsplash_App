## 📜 커밋 컨벤션

```
  - init : 초기화
  - add : 파일 추가
  - feat : 기능 추가
  - delete : 기능 삭제
  - update : 기능 수정
  - fix : 버그 수정
  - refactor: 리팩토링
  - style : 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없음)
  - docs : 문서 (문서 추가(Add), 수정, 삭제)
  - chore : 기타 변경사항 (빌드 스크립트 수정, 에셋 추가 등)
```
